# Hacktoberfest Auto Label Bot

Add label &#39;Hacktoberfest&#39; in new issues.

## Setup

```sh
# Install dependencies
npm install

# Run the bot
npm start
```

## Tests

```sh
npm install --save-dev
npm test
```
